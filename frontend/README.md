# Frontend

    $ cd frontend

## Setup

 1. Install [node 8](https://nodejs.org/en/download/)

 1. Run:

        $ npm install

 1. Copy environment variables from .env.sample:

        $ cp .env.sample .env

## Run

    $ npm start

To exit, press Ctrl-C.
